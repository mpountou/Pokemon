# Pokemon

## Description
A small app that shows all Pokemon species and its details.

<img src="screenshots/app.gif" width="35%"/>

## Features
- Show a list of all Pokemon species
- Show Pokemon's detail

## Tech stack 
- Jetpack compose
- Jetpack navigation
- Dependency Injection with Dagger Hilt
- StateFlow & LiveData
- MVVM architecture
- Retrofit & Gson
- Coil
